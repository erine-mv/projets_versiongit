Exercices JavaScript

1. Addition de deux nombres

Créer une fonction qui prend deux nombres en entrée et retourne leur somme.

Solution :

function addition(a, b) { return a + b; }

2. Multiplication de deux nombres

Créer une fonction qui prend deux nombres en entrée et retourne leur produit.

Solution :

function multiplication(a, b) { return a * b; }

3. Soustraction de deux nombres

Créer une fonction qui retourne la différence entre deux nombres.

Solution :

function subtraction(a, b) { return a - b; }

4. Division de deux nombres

Créer une fonction qui retourne le quotient de deux nombres.

Solution :

function division(a, b) { return a / b; }

5. Trouver le maximum

Créer une fonction qui retourne le plus grand de deux nombres.

Solution :

function max(a, b) { return a > b ? a : b; }

6. Vérifier un nombre pair

Créer une fonction qui détermine si un nombre est pair.

Solution :

function isEven(num) { return num % 2 === 0; }

8. Calculer la puissance

        
Créer une fonction qui retourne le résultat de la base élevée à la puissance de l'exposant.

        
        
        

            
Solution :

            
function power(base, exponent) { return base ** exponent; }

        

        

            
            
            
            

        

    


    
    

        
9. Vérifier une chaîne palindrome

        
Créer une fonction qui détermine si une chaîne de caractères est un palindrome.

        
        
        

            
Solution :

            
function isPalindrome(str) {const reversed = str.split('').reverse().join('');return str === reversed;}

        

        

            
            
            

        

    


    
    

        
10. Somme des nombres dans un tableau

        
Créer une fonction qui retourne la somme des nombres d'un tableau.

        
        
        

            
Solution :

            
function sumArray(arr) {return arr.reduce((acc, num) => acc + num, 0);}

        

        

            
            
            

        

    


    
    

        
11. Nombre aléatoire entre deux valeurs

        
Créer une fonction qui génère un nombre aléatoire entre deux valeurs données.

        
        
        

            
Solution :

            
function randomBetween(min, max) {return Math.floor(Math.random() * (max - min + 1)) + min;}

        

        

            
            
            
            

        

    


    
    

        
12. Trier un tableau

        
Créer une fonction qui trie un tableau de nombres dans l'ordre croissant.

        
        
        

            
Solution :

            
function sortArray(arr) {return arr.sort((a, b) => a - b);}

        

        

            
            
            

        

    


    
    

        
13. Trouver la valeur minimale

        
Créer une fonction qui retourne le plus petit élément d'un tableau.

        
        
        

            
Solution :

            
function findMin(arr) {return Math.min(...arr);}

        

        

            
            
            

        

    


    
    

        
14. Compter les voyelles

        
Créer une fonction qui compte le nombre de voyelles dans une chaîne de caractères.

        
        
        

            
Solution :

            
function countVowels(str) {return str.match(/[aeiou]/gi)?.length || 0;}

        

        

            
            
            

        

    


    
    

        
15. Calculer la factorielle

        
Créer une fonction qui retourne la factorielle d'un nombre.

        
        
        

            
Solution :

            
function factorial(n) {return n === 0 ? 1 : n * factorial(n - 1);}
